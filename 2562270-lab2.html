Latest Project

For my latest project, I developed a Snake game using Java. The game allows the player to control the snake's movements, collect food to grow longer, and avoid running into walls or itself. This project helped me enhance my skills in programming logic, working with 2D arrays, and handling user input in real-time. It also gave me experience with graphical user interfaces (GUI) through the use of the pygame library. I enjoyed tackling the challenge of improving the game's functionality and user experience, and I plan to add more features in the future, like multiple levels and a high-score tracker.
